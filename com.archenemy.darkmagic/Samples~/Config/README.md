# Config Sample

This sample provides one obvious place to configure the V Kit:

- **VConfig.cs** – V (events) trace + example event types
- **SConfig.cs** – S (state machine) settings + example state types
- **WConfig.cs** – W (async runner) trace + guardrails

## Import steps
Unity → Window → Package Manager → select **V** → **Samples** → Import **Config**.
